List of different BIOSes. v3.41

hex_original - EPCS BIOSes from original v3.41 package http://gnogni.altervista.org/
epcs2sdbios - utility to convert EPCS hex file to SD BIOS layout.

These BIOSes don't include JIS2 part, but fully usable. SDBIOS with JIS2 and NEXTOR can be requested on http://gnogni.altervista.org/ site.
